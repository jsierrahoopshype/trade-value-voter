
# HoopsHype Trade-Value Voter (No local setup needed)

Use this repo with **Vercel** + **Supabase**. You don't need to install anything locally.

## Configure
Create two environment variables in your Vercel project:
- `VITE_SUPABASE_URL` = your Supabase Project URL
- `VITE_SUPABASE_ANON_KEY` = your Supabase anon public key

## Deploy
1. Create a GitHub repo (you can upload these files via the GitHub website).
2. Import the repo on Vercel and set the env vars above.
3. Deploy. Use the public URL in your `<iframe>` embed.
